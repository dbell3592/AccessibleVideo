instructions

step one

find a github template to use as the basis for your portfolio page

step two

fork the template to your github repository you will have a copy of the template in your git

step three

open the project folder in visual studio code

step four

clone the url from your github file then paste it into the terminal by typing gitclone followed by the url

step five

you can now begin to edit the html as well as replacing the images i the img file

step six

after completing your necessary changes you can begin to save the changes to github click the source control button to commit the changes and then sync the changes

step seven

to view the changes you made to your project on visual studio code. You must go into the settings section and scroll to GitHub Pages panel. than click on the link

step eight

all done!
